    Why Bool Network?
    Bool Network bridge a robust and fast verification infrastructure by using Dynamic Hidden Committees <DHCs>.
    A DHC functions as a rotational committee comprising a set-up number of DHC nodes randomly elected from all nodes, capable of performing verification and signature tasks as needed by users in aspects including cross-chain asset transfers, the invocation of custodied assets, and Oracle data uploads.
